# 📄 Guia Rápido — Faxineiro Digital (Python)

### 🎯 Objetivo

Mover arquivos de uma pasta para outra automaticamente.

### 📚 O que usar

* `os` → Acessa pastas e arquivos.
* `shutil` → Move arquivos.

### 📝 Passos

1. Defina a pasta de origem.
2. Defina a pasta de destino.
3. Crie a pasta de destino (se necessário).
4. Liste os arquivos.
5. Mova cada arquivo.

### 💻 Comandos importantes

* `os.listdir()` → Lista arquivos.
* `os.makedirs()` → Cria pastas.
* `os.path.join()` → Monta o caminho do arquivo.
* `shutil.move()` → Move o arquivo.

### ✅ Resultado

Arquivos organizados automaticamente, sem precisar mover um por um.
